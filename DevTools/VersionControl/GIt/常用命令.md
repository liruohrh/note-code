# 局域网Clone
- 无需bare仓库，可直接 `git clone ssh://username@host/path/to/repo`
- 如果git仓库在windows，则必须用  `git clone [username@host|hostname]:[driver]:/path/to/repo`


# 修改提交者
```bash
# 查看
git log --format="%h %an <%ae> | %cn <%ce>" -20

# 修改
git rebase --exec 'git commit --amend --reset-author --no-edit' -i xxxx

git rebase --exec 'git commit --amend --reset-author --no-edit' -i eea71f1943c5..a1bedec20886~1
```


#   查看日志
- 某几个：`git log --pretty=format:"%h %ad %an %s" --date=short -90`
- 指定日期之后：`git log --after="2024-01-15"`
- 指定commit之间：`git log ec9aaac47d..HEAD`
#  在clone指定depth时
- 继续拉一些commit下来：`git fetch --deepen=100`
- 拉去从指定时间开始的commit：`git fetch --shallow-since="2026-04-26"`



# 查找指定commit创建时间后的tag

```powershell
$cutoff = [DateTimeOffset]::Parse((git show -s --format="%ci" xxx))
git for-each-ref --sort=creatordate --format="%(creatordate:iso)|%(refname:short)" refs/tags |
ForEach-Object {
    $parts = $_ -split '\|'
    [PSCustomObject]@{
        Date = [DateTimeOffset]::Parse($parts[0])
        Tag  = $parts[1]
    }
} | Where-Object { $_.Date -gt $cutoff } | Sort-Object Date
```

# 查找包含指定commit的tag
