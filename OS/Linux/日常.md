# 输入法
 
 - - **IBus、Fcitx5**：**输入法平台（Input Method Framework）**
    - 负责与操作系统、桌面环境、应用程序通信。
    - 管理输入法切换、候选框显示、快捷键、Wayland/X11 集成等。
    - 它们不是单纯的“输入法库”，更像是输入法运行平台。
- **Rime（中州韵）**：**输入法引擎（Input Method Engine）**
    - 负责把按键转换成候选词。
    - 提供算法、配置系统、方案加载、词典管理等。
    - **Rime 本身不是一种具体的拼音输入法**，而是一个可配置的输入法引擎。
- **雾凇拼音（rime-ice）**：**Rime 输入方案（Schema）**
    - 包含拼音方案、配置、词典、预设等。
    - 本质上是一套 Rime 配置，而不是独立的输入法程序。
    - 它使用 Rime 引擎来工作。
- **词典（Dictionary）**
    - 雾凇拼音自带了多套词库。
    - 用户输入过程中还会生成用户词典，用于学习输入习惯。


# 显示器
## HDMI外接显示器可能会将音频设备切换到显示器

- 先尝试走系统音频设置里调整其他音频设备，如果只显示一个，说明有问题，按下面方式修改

- `wpctl status` ： 查看音频设备，查看音频设备里是否用Speaker（这个是我不外接时走音频设置中显示的音频设备名）
- `pactl list cards`  

```
Card #43
        Name: alsa_card.pci-0000_00_1f.3-platform-skl_hda_dsp_generic
        ...
        Properties:
                device.enum.api = "udev"
                ...
        Profiles:
                off: 关 (sinks: 0, sources: 0, priority: 0, available: yes)
                HiFi (HDMI1, HDMI2, HDMI3, Headphones, Mic1, Mic2): Play HiFi quality Music (HDMI1, HDMI2, HDMI3, Headphones, Mic1, Mic2) (sinks: 4, sources: 2, priority: 10300, available: yes)
                HiFi (HDMI1, HDMI2, HDMI3, Mic1, Mic2, Speaker): Play HiFi quality Music (HDMI1, HDMI2, HDMI3, Mic1, Mic2, Speaker) (sinks: 4, sources: 2, priority: 10200, available: yes)
                pro-audio: 专业音频 (sinks: 5, sources: 3, priority: 1, available: yes)
        Active Profile: HiFi (HDMI1, HDMI2, HDMI3, Headphones, Mic1, Mic2)
```
- Profiles: 音频设备
	- "HiFi (HDMI1, HDMI2, HDMI3, Headphones, Mic1, Mic2)"
	- "HiFi (HDMI1, HDMI2, HDMI3, Mic1, Mic2, Speaker)"
- Active Profile：激活设备



- 切换Profile为 Speaker
`pactl set-card-profile alsa_card.pci-0000_00_1f.3-platform-skl_hda_dsp_generic "HiFi (HDMI1, HDMI2, HDMI3, Mic1, Mic2, Speaker)"`
- 之后插拔HDMI依然保持这个配置

# 声卡

- **Card**：一块物理音频硬件（如 Intel 声卡、USB 声卡、蓝牙耳机）。
- **Profile**：这块声卡当前启用了哪些功能和接口，相当于"工作模式"。
- Port：对应音视频输入输出端点（不是物理接口）
	- Headphones：耳机控
	- Speaker：内置音频
	- HDMIn：HDMI接口，还会显示如device.product.name = "U270CA"（我的显示器品牌）
	- Stereo Microphone：耳机孔麦克风
	- Digital Microphone：内置麦克风
- **Sink**：一个实际的声音输出目标（扬声器、HDMI 音频等），播放器把声音送到这里。
- **Source**：一个实际的声音输入目标（麦克风等）。
- **Link**：把应用程序（Chrome、VLC 等）连接到某个 Sink 或 Source 的"线路"。
- **HDMI1 / HDMI2 / HDMI3**：通常不是 3 个物理 HDMI 接口，而是 GPU 提供的多个数字音频输出通道；只有连接了显示器的那个通道会显示为 `available`。

- **`wpctl`**：面向 PipeWire/WirePlumber 的原生管理工具。
- **`pactl`**：通过兼容层访问同一个 PipeWire 音频服务器。

- `wpctl status` ： 查看音视频设备状态
- `pactl list cards`  ： 查看音视频设备
- `pactl list sink`： 查看当前profile打开的sink
