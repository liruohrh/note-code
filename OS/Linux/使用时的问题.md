# 不知道网络为什么卡死在第二个无法访问的DNS服务器114.114.114.114
```
nmcli dev show enp3s0 | grep DNS 
IP4.DNS[1]: 8.8.8.8 
IP4.DNS[2]: 114.114.114.114
```


## 解决方法
- `sudo nano /etc/systemd/resolved.conf`
```
DNS=8.8.8.8 1.1.1.1
FallbackDNS=
```
---
- `sudo systemctl daemon-reload`
- `sudo systemctl restart systemd-resolved`
