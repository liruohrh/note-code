- 查询时一起查询关联Preload
- 关联操作Association
	- Replace：顾名思义，将关联完完全全替换为当前传递的，但对于旧数据不会进行删除而是只是取消旧关联
		- OneToMany
			- `INSERT && ON DUPLICATE KEY UPDATE`关联 
				- 默认只是更新关联字段，FullSaveAssociations=true才会全更新
			- 设置关联表的关联字段为null
				- 替换不会删除不在当前关联值中的
		- ManyToMany
			- `INSERT && ON DUPLICATE KEY UPDATE` 关联 
			- `INSERT && ON DUPLICATE KEY UPDATE` 中间表
			- DELETE中间表中不在当前关联值中的
